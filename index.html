Clouds, Servers and Services

Cloud computing is a service that enables, efficient on-demand network access to a shared pool of computing resources through the internet with a pay-as-you-go pricing. Cloud computing provides essential, flexible and low-cost IT resources. Cloud computing it sort of an outsource platform, therefore users may not worry about managing the hardware. Users are provided with resources, they need, instantly and only pay when is use.

Software as a service(SAAS) enables users to access application software’s independently of the platform on an on demand pay-per use basis without even installing the software in their computer, cloud operates a single software, which will be accessible for multiple users on the network which is efficient and cheap. The vendor will manage and be responsible for all computing resources, leaving users with less worrying about upgrade or any other software issues. The service will be accessible to users via the web browsers or an application platform provided by the vendor. The intended users for this service are end users of softwares. There are few service providers such as, Google Docs, Microsoft Office 365, HR Helpdesk, etc. SAAS is accessible from any platform that has access to the internet. This enable multiple tenancy and is very useful for collaborative working. Other than that, some vendors offers simple and reliable software tools. However, there will be transportability and bowser issues because the performance of the internet may have an influence on the accessibility of it. So, if there is no internet connection, user will not be able to use it. Furthermore, SAAS might have some restrictions on some aspects of the service.

Platform as a service (PASS) is another service, but it provides a platform to enable users, mainly developers to build and manage their applications and services. This platform is composed of programming language execution environment, an operating system, web server and a database. It delivers a setting that allow users, mainly programmers and software developers to build, compile and run their work without any inconvenience. Users only concern is to manage the data and application source. All other resources are handled and managed by the vendor. Overall the PAAS is an environment for software developers. The intended users for this service are programmers and developers. Some examples of service providers are Amazon Web Service, Google App Engine, Windows Azure, etc. PAAS might be cost efficient but it only offers limited amount of languages and tools. Other than that, it helps developers promote their software and make the development easier by exposing it to the market.

Design and structure are provided by infrastructure as a service (IAAS) It delivers all computing resources such as data storage, virtualization, servers and networking in a virtual environment therefore many users will be able to gain access and use of this service. Users will only be responsible operating resources such as applications, data, runtime and middleware. The intended users for this service are system admins. Amazon Ec2, Go Grid, Rackspace.com are examples of service provider for IAAS. IAAS is very flexible, it might due to the infrastructure being provided by cloud which. It also helps improve in scalability. However, the network and service will experience some delay if the internet connection is bad or there is a cloud upgrade.
